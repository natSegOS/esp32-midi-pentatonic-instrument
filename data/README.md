# Data

This directory contains inter-onset interval (IOI) samples collected during development.

## Files
- `ioi_120bpm_run_01.csv`: keyboard-collected tap data from a steady ~120 BPM run

## Notes

The dataset was collected on the host machine to isolate human timing variability from microcontroller transport effects.

This data is used as the empirical basis for the computational stress tests in `analysis/`
